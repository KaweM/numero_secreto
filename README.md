# 🎯 Número Secreto

Projeto simples de um jogo onde o usuário deve adivinhar um número secreto gerado pelo sistema.

## 📌 Sobre o projeto

Este projeto foi desenvolvido como prática de lógica de programação durante meus estudos na Alura.

A proposta é criar um jogo interativo onde o usuário tenta acertar um número aleatório, recebendo dicas se o valor é maior ou menor até acertar.

## 🚀 Funcionalidades

- Geração de número aleatório  
- Validação de tentativas  
- Dicas para o usuário (maior ou menor)  
- Contagem de tentativas  

## 🛠️ Tecnologias utilizadas

- HTML  
- CSS  
- JavaScript
